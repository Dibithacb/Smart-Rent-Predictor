Smart Rent Predictor (HomeWorth) 
Description: Real estate rental platform with intelligent rent estimation based on location and 
property features. 
Core Features: 
● Map view of rental listings 
● Property cards with images, details, and pricing 
● Advanced filter options (location, bedrooms, amenities, price range) 
● Property detail page with gallery and specifications 
● Comparison tool for multiple properties (side-by-side) 
● Neighborhood information display 
● Saved searches and favorites 
● Sort by price, popularity, rating
  Responsive UI for Desktop and Mobile

Tech Stack
Framewor: React
Styling: Tailwind
State Management: Context
Routing: React Router
